En la unidad 3 empezamos a trabajar con los controladores de versiones, en este caso Git y GitHub como almacenamiento para nuestros repositorios de programas que vayamos a crear y también como herramienta que todo programador debe aprender para poder llevar a cabo su trabajo. Estas herramientas resultan muy útiles cuando formamos grupos de trabajos de multiples desarrolladores para que cada uno puedo aportar su avance de un mismo proyecto y poder mantener un historial de todos los cambios que se han realizado en el mismo.
De la misma manera podemos mantener multiples ramas que salen del mismo proyecto para diferentes objetivos, tanto como producción, pruebas, nuevos lanzamientos o características que se agregarán, etc.
